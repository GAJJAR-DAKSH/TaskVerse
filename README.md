# 🚀 TaskVerse

**TaskVerse** is a modern and responsive task management web application that helps users organize daily activities efficiently. Built with HTML, CSS, JavaScript, Bootstrap, and Local Storage, it provides a simple yet powerful interface for managing tasks and boosting productivity.

## ✨ Features

- ➕ Add new tasks
- 📝 Task title and description support
- ✅ Mark tasks as completed
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks
- 💾 Data persistence using Local Storage
- 🎨 Responsive and modern UI
- ⚡ Fast and lightweight
- 📱 Mobile-friendly design
- ⌨️ Enter key support for quick task addition
- 🔄 Dynamic updates without page refresh

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 4
- Local Storage API
- Typed.js
